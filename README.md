# 📧 Spam Message Detection using Machine Learning

## 🚀 Project Overview
This project detects whether a message is **Spam or Not Spam (Ham)** using Machine Learning and Natural Language Processing.

## 🧠 Tech Stack
- Python
- Pandas
- Scikit-learn
- Streamlit
- Naive Bayes Algorithm

## 📊 Model Accuracy
Achieved accuracy: **~98%**

## 📂 Project Files
- app.py → Streamlit web app
- task1.py → Model training code
- spam_model.pkl → Trained model
- vectorizer.pkl → Text vectorizer
- spam.csv → Dataset

## ▶️ How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
